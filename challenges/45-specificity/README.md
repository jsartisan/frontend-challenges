Given the provided HTML and CSS code snippets below, what would be the color of the text 'Guess my color'?

```html
<body>
  <div class="outer">
    <p class="inner">Guess my color</p>
  </div>
</body>
```

```css
p.inner {
  color: purple;
}

.outer p {
  color: red;
}

.inner {
  color: blue;
}

p {
  color: green;
}
```
